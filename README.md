#Espresso Corner#
Meteor app with angular and material design

##Getting started##
###Install Meteor###
`curl https://install.meteor.com/ | sh`
https://www.meteor.com/install

###Download dependencies###
`meteor install`
`npm install`

###Run app###
`meteor`
